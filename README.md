# Nome do projeto: DOOM GALÁTICO 
 O  Doom Galático é inspirado no jogo de tiro Doom '97 e Wolfeinstein 3D, de modo a criar um unverso de galáxia em um cenário que fugisse com os esteriótpos acerca de jogos de tiro. Mas que ao mesmo tempo reelembrasse os velhos tempos do jogo inspirado.

 
 Para jogar : Use W, A, S, D para se mover pelo mapa e explorar o ambiente em primeira pessoa.
Mire com o mouse e clique com o botão esquerdo para atirar.Na qual o objetivo final é eliminar inimigos e não levar dano o suficiente para ser morto, explorando os corredores temáticos.
